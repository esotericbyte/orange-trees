## About orange-trees 🍊

orange-trees is an implementation of the **Tree** data structure. It comes with a rich set of methods to manipulate and query nodes.

---

## Get started 🏁

### Add orange-trees to your Cargo.toml 🦀

```toml
orange-trees = "0.1.0"
```
