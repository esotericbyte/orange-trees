## About lemon-tree 🍋

lemon-tree is an implementation of the **Tree** data structure. It comes with a rich set of methods to manipulate and query nodes.

---

## Get started 🏁

### Add lemon-tree to your Cargo.toml 🦀

```toml
lemon-tree = "0.1.0"
```
